# Category Parser - Chrome Extension

Расширение для парсинга категорий и подкатегорий с веб-сайтов.

## Установка

1. Откройте `create_icons.html` в браузере и скачайте иконки в папку `icons/`
2. Откройте `chrome://extensions/` в Chrome
3. Включите "Режим разработчика"
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку с расширением

## Использование

1. Откройте сайт с категориями
2. Нажмите на иконку расширения
3. Нажмите "Спарсить категории"
4. Экспортируйте данные в JSON или CSV
